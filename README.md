# SkinSense

SkinSense merupakan aplikasi berbasis web yang dirancang untuk membantu pengguna mengenali jenis kulit wajah serta memperoleh rekomendasi perawatan kulit (skincare) yang sesuai. Sistem memanfaatkan model machine learning yaitu Teachable Machine, untuk melakukan klasifikasi jenis kulit berdasarkan gambar wajah yang diunggah pengguna, kemudian memberikan rekomendasi produk dan langkah perawatan yang disesuaikan dengan hasil deteksi.

Aplikasi ini mendukung beberapa kategori jenis kulit, yaitu Normal, Berminyak (Oily), Kering (Dry), dan Berjerawat. Setiap jenis kulit memiliki penjelasan karakteristik serta rekomendasi produk yang mencakup tahapan dasar perawatan, seperti pembersih wajah, toner, pelembap, dan tabir surya.

SkinSense dikembangkan menggunakan teknologi web modern dengan antarmuka yang sederhana dan mudah digunakan, sehingga dapat membantu pengguna memahami kondisi kulit mereka serta memilih rutinitas perawatan yang lebih tepat sesuai kebutuhan.
